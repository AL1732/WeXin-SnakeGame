# 抖音多人版贪吃蛇游戏

一个基于抖音平台的多人在线贪吃蛇游戏，使用现代全栈技术栈开发，从单机原型逐步演进到多人全战上线。

## 项目概述

这是一个完整的多人在线贪吃蛇游戏项目，支持实时多人对战、房间匹配、道具系统、排行榜等功能，并最终将在抖音平台上线。

## 功能特性

- 🎮 经典蛇形游戏玩法
- 🌐 多人在线实时对战
- 🏠 房间创建与匹配系统
- 🎯 道具系统（加速、减速、穿墙）
- 🏆 实时排行榜
- 📱 响应式设计，支持不同设备
- 🎨 现代化的视觉设计
- ⌨️ 键盘和触摸屏控制
- 📤 抖音分享功能
- 👥 好友排行榜
- 🏅 成就/奖励系统

## 开发计划

### 阶段 1：单机原型
- **目标**：完成单人贪吃蛇核心玩法
- **技术点**：React + Canvas + TypeScript
- **输出**：可玩的单机游戏，蛇吃食物、碰撞死亡

### 阶段 2：多人原型
- **目标**：支持多人同屏对战
- **技术点**：Node.js + WebSocket + TypeScript
- **输出**：两个玩家可以同屏对战，实时位置同步

### 阶段 3：房间匹配 & 排行榜
- **目标**：完善房间系统和排行榜
- **技术点**：Redis + MySQL + REST API
- **输出**：房间匹配、分数统计、排行榜可用

### 阶段 4：道具 & 特殊技能
- **目标**：丰富游戏玩法
- **技术点**：Node.js + 前端逻辑 + Redis缓存
- **输出**：游戏玩法丰富，可测试策略互动

### 阶段 5：容器化部署 & CI/CD
- **目标**：部署到阿里云 ECS
- **技术点**：Docker + CI/CD + 阿里云 ECS
- **输出**：可快速迭代上线

### 阶段 6：数据分析 & 日志监控
- **目标**：监控游戏数据和性能
- **技术点**：Grafana + Kafka + Redis
- **输出**：可分析游戏数据，优化玩法与服务器性能

### 阶段 7：抖音 SDK 接入
- **目标**：抖音平台上线
- **技术点**：抖音 SDK
- **输出**：小游戏可以在抖音上线，其他用户可搜索玩

## 控制方式

### 触摸屏控制
- **滑动手势**：在屏幕上滑动控制蛇的移动方向
- **点击屏幕**：点击屏幕任意位置暂停/继续游戏
- **双击屏幕**：重新开始游戏
- **道具使用**：点击屏幕上的道具图标使用对应道具

## 游戏机制

### 核心玩法
- **蛇的移动**：每帧更新蛇的位置，根据方向键输入改变移动方向
- **食物生成**：随机在画布上生成食物，确保不会出现在蛇身上
- **碰撞检测**：检测蛇是否撞到墙壁或自身
- **分数计算**：每吃一个食物增加分数，蛇的长度增加
- **游戏速度**：随着分数增加，游戏速度逐渐提高

### 多人玩法
- **房间系统**：创建房间或加入好友房间
- **随机匹配**：自动匹配其他玩家
- **实时同步**：WebSocket 实时同步玩家位置
- **排行榜**：实时更新玩家分数排名

### 道具系统
- **加速**：暂时提高移动速度
- **减速**：暂时降低其他玩家速度
- **穿墙**：暂时可以穿过墙壁

## 抖音小游戏上线流程

### 1. 准备工作
- **注册抖音开发者账号**：访问 [抖音开放平台](https://open.douyin.com/) 注册开发者账号
- **创建小游戏应用**：在开放平台创建新的小游戏应用，获取 AppID
- **完善应用信息**：填写游戏名称、描述、图标等基本信息

### 2. 开发与测试
- **集成抖音 SDK**：在前端项目中集成抖音小游戏 SDK
- **本地测试**：使用抖音开发者工具进行本地测试
- **真机测试**：在真实设备上测试游戏功能

### 3. 提交流审
- **准备提审材料**：
  - 游戏安装包（.zip格式）
  - 游戏截图（至少3张）
  - 游戏视频（15-30秒）
  - 游戏说明文档
- **提交审核**：在开放平台提交游戏审核
- **审核反馈**：根据审核意见进行修改

### 4. 上线运营
- **审核通过**：游戏审核通过后，设置上线时间
- **灰度发布**：先进行小范围灰度测试
- **全量发布**：灰度测试通过后，全量上线
- **运营监控**：通过抖音开放平台监控游戏数据

### 5. 注意事项
- **合规要求**：确保游戏内容符合抖音平台规范
- **性能优化**：保证游戏在各种设备上流畅运行
- **用户体验**：优化游戏界面和操作体验
- **数据安全**：保护用户数据安全

### 6. 推广与运营
- **抖音内推广**：利用抖音平台的推广工具
- **社交媒体宣传**：通过社交媒体宣传游戏
- **活动运营**：定期举办游戏活动，增加用户粘性
- **数据分析**：根据数据反馈优化游戏体验
